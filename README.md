Struktur Singkat Repository
--------------------------
- `JS01/` sampai `JS15/`: folder per pertemuan yang berisi notebook (LAT = Latihan, TG = Tugas, UTS/KUIS = evaluasi).
- `data/`, `dataset/`, `images/`, `others/`: dataset yang digunakan pada beberapa pertemuan (Iris, Titanic, Churn, voice, dan lain-lain).
